# Images Directory

This directory contains all images for Ashley Fields' website.

## Image Guidelines

### Product Images
- **Size**: 800x800px recommended (minimum 600x600px)
- **Format**: WebP preferred, JPG/PNG acceptable
- **Quality**: High quality, well-lit photos
- **Background**: Clean, neutral backgrounds
- **Naming**: Use descriptive names (e.g., `ceramic-vase-sage.jpg`)

### Site Images
- **Hero images**: 1200x800px landscape
- **About images**: 600x600px square
- **Gallery images**: 400x400px square
- **Favicon**: 32x32px ICO format

### Image Optimization
1. Compress images for web
2. Use appropriate formats (WebP for photos, PNG for graphics)
3. Include alt text in HTML
4. Use lazy loading for better performance

### Adding New Images
1. Place images in this directory
2. Update HTML files to reference new images
3. Include descriptive alt text
4. Optimize for web use
